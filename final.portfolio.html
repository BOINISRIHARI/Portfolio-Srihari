Recent Projects

Project 1

Predictive analytics for ventilation pressure forecasting in clinical setting.

Involves using historical patient data and machine learning models to predict the optimal pressure a mechanical ventilator should apply to support a patient's breathing. By analyzing patterns in vital signs, lung mechanics, and previous ventilation data, the system can forecast future pressure needs in real-time, helping to adjust ventilator settings more accurately and safely.

Project 2

image classification of abnormal red blood cells using decision tree learning

Image classification of abnormal red blood cells using decision tree learning involves analyzing microscopic images of blood samples to automatically identify and categorize red blood cells based on their shape, size, or texture. A decision tree algorithm learns from labeled training images by creating a flowchart-like model where each decision node represents a feature (like color intensity or edge shape), and each branch leads to a classification outcome (e.g., normal, sickle-shaped, or elliptocyte).
